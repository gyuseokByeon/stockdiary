# stockdiary
주식종목 발굴 웹사이트

### 주식검색 사이트.
django 를 이용한 주식검색 사이트를 개발했습니다. 
beautifulsoup4 를 이용해서 네이버증권 사이트의 정보를 수집하여 매수 추천 종목을 찾아줍니다.
